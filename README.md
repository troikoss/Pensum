Pensum is a Windows-style Task Manager for Android, built specifically for power users in desktop environments.

Note: This project is semi-vibecoded.

## Features:
Real-time Monitoring: Live updates of CPU and RAM consumption across system and user processes.

Grouped View: Organized categorization of Apps, Background processes, and Android system processes.

Desktop Mode Optimized: Built to run in windowed mode with a responsive layout for desktop-style multitasking with full keyboard and mouse support.


## Planned Features:

Resource History: Real-time system resource graphs (CPU/RAM history).


## How to install:

1- Install the app from the [Releases](https://github.com/troikoss/Pensum/releases).

2- Install [Shizuku](https://shizuku.rikka.app)

3- Follow Shizuku setup process.

4- Grant Pensum permission and restart Pensum.
